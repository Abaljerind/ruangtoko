## Overview

### The challenge

Users should be able to:

- Browse the homepage and view product highlights (Hero section & featured items)
- View all products and browse by category
- Open a product detail page to see more information
- Search and filter products using frontend-based filtering
- Add products to cart and persist cart data using `localStorage`
- View, update, or remove items from the cart page
- Continue to checkout using a dummy UI
- View their past orders in the **Order History** page
- Click an order to see complete **Order Details**
- Use the website smoothly on all screen sizes (mobile & desktop)
- Experience a clean and organized UI with reusable component architecture
